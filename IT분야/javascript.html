3. 코딩의 기본 operator, if, for loop 코드리뷰

1. string concatenation

문자열과 문자열을 합하여 새로운 문자열을 만들수 있다.

2. numeric operators

+, -, /, *, %, ** 등의 연산자가 있다.

3. ++, -- operators

++부호가 앞에 있을 경유 = preIncrement = 1을 더한다.

변수다음에 ++일경우 = posrIncrement = 변수의 값을 posrIncrement에 할당후 변수에 1을 증가 한다.

4. = operator

할당이 가능한 operator

5. <= operator

작거나 같거나 크거나 같거나 등 여러가지 상황일때 사용

6. Logical operators

(or): 3개중에 하나라도 true가 있다면 true로 계산한다. *사용시 연산을 적게하는 것들을 앞에둬야 한다.

(and): 3개중에 하나라도 false가 있다면 false로 계산하다.

(not): 값을 반대로 변경한다. true 일경우에 false로 변경

7. Equality operators

==: loose Equality → type을 변경하여 검사한다.

===: strict Equality → type이 다를시 false로 계산된다. 보통 이함수를 이용하여 검사하는 것이 좋다.

8. if operators

if: 조건문으로 if가 true 이거나 false로 설정후 block을 실행한다. else if 는 다른 조건문을 추가한다. else는 위조건에 해당하지 않을경우 실행된다.

ternary operators: 간단하게 출력할경우 사용한다. ex) 'beom'?'yes':'no' 범일경우 yes를 출력 아닐경우 no를 출력한다.

swith operators: switch 안에 값이 ~이면 ~을 실행한다. (여러가지 조건이 필요할때 else if 를 간단하게 해줄수 있음)

while loop: 반복문 으로 조건에 맞을때 까지 반복을 한다.

do-while loop: 먼저 블럭을 실행후 조건이 맞는지 검사 한다.

for loop: begin → condition → step 순으로 진행된다.

nested loop: for문 안에 for문이 있는 형식이다.

4. Arrow Function은 무엇인가? 함수의 선언과표현!

1. Function

함수의 이름을 지정하고 파라미터를 쭉 나열한 다음 함수안에 요소들을 작성후 리턴을 한다.하나의 함수는 한가지의 일만 하게 만들어야 한다. 함수의 이름을 지정할때는 동사의 형태로 만든다. 유용한 함수를 만들려면 parameter 로 메세지를 전달 한다. 자바스크립트에서는 type이 없어서 문자를 전달하는지 숫자를 전달하는지 알수 없다. 타입스크립트에서는 파라미터나 리턴타입에 타입을 명시해야 한다.

2. parameters

premitive 타입은 메모리에 value가 저장되어있어 value가 전달 // object는 reference가 저장되어있어 reference를 전달한다. object는 reference로 전달되기 때문에 함수안에서 오브젝트의 값을 변경하게 되면 그 값의 그대로 메모리에 저장

3. Default Parameters

showMessage(message, from)// 파라미터의 값이 전달되지 않을경우 unknown 으로 대채된다.

4. Rest Parameters

(...args) 배열형태로 전달된다.

5. Return

return 값이 없는것들은 (return undefined;) 값이 숨어져 있다. ->(생략 가능) 다른값을 return 할수 있다.

6. Early Return

if 문에서 조건이 맞지 않을때 빨리 return 하고 조건이 맞을때만 실행

7. Function Expression

이름이 없는 함수 : anonymous function

이름이 있는 함수 : named function

a function declaration can be called earlier than it is defiend. (hoisted) == var

a function expression is created when the execution reaches it.

8. Callback hell

상황이나 조건에 맞게 함수를 부르는 것. ex) if,else

9. Arrow Function

이름이 없는 함수이다. simple하게 만들어 준다. ex) const add = (a, b) => a+b;

10. IIFE

함수를 선언함과 동시에 (로 묶을시) 바로 실행 가능하다.
