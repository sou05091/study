HTML 공부일지 - 참고 사이트 (생활코딩)

start study- January 01, 2021

저같은 경우에는 순서를 조금 다르게 하여서 공부를 시작하였습니다. 수업강의를 먼저 듣기 보다는 먼저 웹사이트를 만들고 그다음 차근차근 기초를 쌓아 나갔습니다. 물론 주위 상황상 아무것도 모르는 상태에서 웹사이트를 먼저 만들어야 하였지만 오히려 공부에는 더 큰 도움이 되었던 것 같습니다.

생활코딩님의 수업을 참고로 정리해 보았습니다. 나름 html은 쉬운 언어에 속하여서 간단하게 정리만 해보았습니다.

위 사진은 html에서 사용되는 언어들입니다. 중요한 부분들만 간단하게 설명만 하고 넘어가겠습니다.

head: 본문설명을 위해 존재

body: 본문을 적음

strong: 글에 강조를 할때 사용한다

u: 글 밑에 줄이 생긴다. underline

h1: 머릿말 제목을 표시한다.

br: 줄을 변경할때 사용한다.

p: 이또한 줄을 변경할때 사용하지만 br 보다는 p를 많이 선호하는 편이다. 이유는 p를 사용할 경우 단락의 간격이 고정이 되기 때문이다.

img: 이미지를 추가할때 사용된다. 이미지 위치를 정해줘야지 이미지가 나타난다

li: 리스트로 나타내고 싶을때 사용된다.

ul: 리스트와 같이 사용되며 묶음 표시를 한다.

meta: 예시) meta charset="utf-8" 이 의미는 utf-8로 만들어진 웹사이트를 utf-8로 열어라 라는 뜻으로 글깨짐을 방지한다.

a: 링크테그로 글에 링크를 걸때 사용된다.
